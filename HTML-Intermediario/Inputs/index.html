HTML Intermediário

Inputs

Input Text

<input type="text" name="nome" value="Lucas" id="nome" placeholder="Digite seu nome aqui" >

  • O atributo type é o tipo desse input
  • O atributo name é o link para onde esse input vai ser armazenado
  • O atributo value é o que vai dizer se esse input tem um valor padrão
  • O atributo id é um identificador para esse input
  • O atributo placeholder é um texto informativo do que deve ser colocado nesse input

Input number, password, Submit

<input type="number">

Label

Atributo for indica a qual é o campo de input que esse label pertence. Deve ser correspondente ao id do input

<label for="numero">Number</label>
