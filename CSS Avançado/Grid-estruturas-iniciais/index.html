  1. Defina o container com display grid
    No Display grid os elementos ocupam todo o espaço do container
    No Display inline-grid eles ocupam o espaço do maior elemento
  2. grid-template-colums para definir como vai ser a colunas
    ex: grid-template-columns: 100px auto; --> Cria uma coluna com 100px e outra preenchando todo o resto de espaço do container
  3. grid-template-rows: 150px 50px; para definir o tamanho das linhas, nesse exemplo 150 na primeira, 50 na segunda e as outras o tamanho padrão
    Para repetir o mesmo tamanho para outras linhas sem ter que ficar digitando a mesma coisa:
    grid-template-rows: repeat(2, 100px) --> sendo o primeiro parâmetro quantas linhas vão ter aquele tamanho
  4. Abreviando essas duas propriedades:
    grid-template: repeat(2, 150px ) / 100px auto;
  5. grid-gap: 10px; --> para separar as linhas e colunas
    grid-column-gap: 10px; --> separa só as colunas
    grid-row-gap:30px; -->para separar só as linhas
    Se passar 2 valores no grid-gap o primeiro é para linhas e o segundo para colunas
